# Post-Pilot Review Template

## Purpose

This template should be copied after a controlled pilot ends.

It helps the team decide what to fix, what to defer, and whether the MVP is ready for a wider rollout.

```text
## Support Lessons

## Product Lessons

- lesson 1

## Architecture Lessons

## Must Fix Before Wider Rollout

- item 1

## Should Fix Soon

- item 1

## Future Backlog

## Decision

Choose one:

- proceed to wider pilot
- run another controlled pilot
- stabilize further before more users
- revisit MVP scope

```

## Final Rule

The post-pilot review should be honest and actionable. It should protect the product from scaling problems too early.
