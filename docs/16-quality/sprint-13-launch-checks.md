# Sprint 13 Launch Checks

## Purpose

This document defines checks for Sprint 13.

Sprint 13 prepares launch-day execution and immediate post-launch operations.

## Check 1: Scope

Sprint 13 may include:

- launch runbook updates
- monitoring improvements
- support readiness updates
- release decision notes
- rollout reporting updates
- small stability fixes required for launch confidence
- documentation updates required by launch operations

Sprint 13 should not include:

- broad new modules
- unrelated product expansion
- public marketing features inside product code
- large refactors unrelated to launch safety

## Check 2: Launch Decision

Confirm:

- release decision note exists
- release reference is known
- known limitations are documented
- release owner is identified
- support owner is identified

## Check 3: Runbook Readiness

Confirm:

- public launch runbook exists
- release reference fields are clear
- backup confidence is reviewed
- support channel is active
- monitoring approach is clear

## Check 4: Verification

Confirm basic MVP flows were checked:

- app startup
- user sign-in
- organization selection
- Religious dashboard
- member workflow
- visitor workflow
- attendance workflow where implemented

## Check 5: Support Readiness

Confirm:

- support owner is available
- support process is active
- known limitations are easy to explain
- issue capture process is ready

## Check 6: Monitoring Readiness

Confirm:

- post-launch monitoring playbook exists
- first-hour review is defined
- first-day review is defined
- status options are defined
- next action process is clear

## Check 7: Decision After Launch Window

Choose one:

```text
continue launch
continue with limitations
watch closely
pause expansion
needs fix
```

## Check 8: Documentation

Confirm docs were updated where launch operations changed setup, support, monitoring, or release decision direction.

## Final Rule

Sprint 13 is complete when launch-day execution can happen with a clear runbook, support owner, monitoring plan, and written release decision.
