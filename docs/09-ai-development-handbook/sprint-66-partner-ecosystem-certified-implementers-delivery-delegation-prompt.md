## Expected Output

The coding agent should produce a focused partner ecosystem foundation for partner organizations, certified implementer profiles, certification badges, delivery assignments, delegation requests, scoped access placeholders, implementation links, blueprint/package authorization links, quality placeholders, and audit-safe partner operations.

## Final Rule

Partner delivery must extend implementation capacity without weakening tenant control. Delegated access must be explicit, scoped, time-aware, permission-protected, auditable, and revocable by reviewed process.
